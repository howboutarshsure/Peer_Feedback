Vector Class Operations

Introduction to Vector Class

The Vector class is a resizable array that can hold elements of any data type. It dynamically adjusts its capacity as elements are added or removed.

Binary Search

Binary Search is a highly efficient search algorithm that uses a divide-and-conquer approach. It works on sorted arrays by repeatedly dividing the search interval in half. If the value of the search key is less than the item in the middle of the interval, narrow the interval to the lower half. Otherwise, narrow it to the upper half. The algorithm has a time complexity of O(log n).

Pseudo Code

Binary Search:
    function binarySearch(arr, item):
        return recursiveBinarySearch(arr, item, 0, length(arr) - 1)

    function recursiveBinarySearch(arr, item, low, high):
        if high < low:
            return -1
        mid = low + (high - low) / 2
        if arr[mid] == item:
            return mid
        else if arr[mid] > item:
            return recursiveBinarySearch(arr, item, low, mid - 1)
        else:
            return recursiveBinarySearch(arr, item, mid + 1, high)

Time and Space Complexity

Time Complexity: O(log n). Binary Search halves the search space with each iteration, making it very efficient for large datasets.

Space Complexity: O(1). Binary Search uses a constant amount of extra space for variables such as low, high, and mid. If implemented recursively, the space complexity can become O(log n) due to the call stack.

Iterator Design Pattern

The Iterator pattern allows sequential access to the elements of a data structure without exposing its underlying representation. This pattern is commonly used with collection-type classes, such as lists, stacks, and queues.

Pseudo Code

Iterator Pattern:
    class VectorIterator:
        function __init__(data, count):
            self._data = data
            self._count = count
            self._position = -1

        function current():
            if self._position == -1 or self._position >= self._count:
                return null
            return self._data[self._position]

        function moveNext():
            if self._position < self._count - 1:
                self._position += 1
                return true
            return false

        function reset():
            self._position = -1

    class Vector:
        function getEnumerator():
            return VectorIterator(self._data, self._count)

Time and Space Complexity

Time Complexity: O(1) per operation. Each operation (moveNext, current, reset) in the iterator runs in constant time.

Space Complexity: O(1). The iterator only needs a few variables to keep track of the current position and reference to the data array.
