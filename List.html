Doubly Linked List Operations

Introduction to Doubly Linked Lists

A doubly linked list is a linear data structure consisting of nodes where each node contains a data record and two pointers: one to the next node and one to the previous node. This allows traversal in both directions.

Structure of a Doubly Linked List

The doubly linked list uses auxiliary nodes: the Head and the Tail, which simplify implementation and are always maintained, even when the list is empty.

Node {
    Value
    Next
    Previous
}

Basic Operations

AddFirst

Adds a new node containing the specified value at the start of the doubly linked list. This involves creating a new node and adjusting the pointers of the Head and the next node.

function AddFirst(value):
    node = new Node(value)
    node.Next = Head.Next
    node.Previous = Head
    Head.Next.Previous = node
    Head.Next = node

AddLast

Adds a new node containing the specified value at the end of the doubly linked list. This involves creating a new node and adjusting the pointers of the Tail and the previous node.

function AddLast(value):
    node = new Node(value)
    node.Previous = Tail.Previous
    node.Next = Tail
    Tail.Previous.Next = node
    Tail.Previous = node

RemoveFirst

Removes the node at the start of the doubly linked list. This involves adjusting the pointers of the Head and the next node to exclude the first node.

function RemoveFirst():
    if Head.Next == Tail:
        throw "List is empty"
    node = Head.Next
    Head.Next = node.Next
    node.Next.Previous = Head
    node = null

RemoveLast

Removes the node at the end of the doubly linked list. This involves adjusting the pointers of the Tail and the previous node to exclude the last node.

function RemoveLast():
    if Tail.Previous == Head:
        throw "List is empty"
    node = Tail.Previous
    Tail.Previous = node.Previous
    node.Previous.Next = Tail
    node = null

Other Operations

AddBefore

Adds a new node before the specified node. This involves creating a new node and adjusting the pointers of the given node and its previous node.

function AddBefore(node, value):
    if node == null:
        throw "Invalid node"
    newNode = new Node(value)
    newNode.Previous = node.Previous
    newNode.Next = node
    node.Previous.Next = newNode
    node.Previous = newNode

Visualize AddBefore Operation

AddAfter

Adds a new node after the specified node. This involves creating a new node and adjusting the pointers of the given node and its next node.

function AddAfter(node, value):
    if node == null:
        throw "Invalid node"
    newNode = new Node(value)
    newNode.Next = node.Next
    newNode.Previous = node
    node.Next.Previous = newNode
    node.Next = newNode

Visualize AddAfter Operation

Remove

Removes the specified node from the doubly linked list. This involves adjusting the pointers of the previous and next nodes to exclude the given node.

function Remove(node):
    if node == null:
        throw "Invalid node"
    node.Previous.Next = node.Next
    node.Next.Previous = node.Previous
    node = null
